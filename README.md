# 🛍️ StyleBay

StyleBay is an **ASP.NET Core MVC web application** developed with separate **Admin** and **User** modules for managing an online shopping platform.

---

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Browse Products
- Product Details View
- Add to Cart
- Order Management
- Responsive UI

### 🛠️ Admin Module
- Admin Login
- Manage Products
- Add / Edit / Delete Products
- Manage Users
- Manage Orders
- Dashboard Overview

---

## 🛠 Tech Stack

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML5
- CSS3
- JavaScript
- Bootstrap

---

## 🗄 Database

- SQL Server Database
- Entity Framework Migrations Used
